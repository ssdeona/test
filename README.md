Блок 1. Валидация корректных email-адресов
1. Название: Ввод корректного email в стандартном формате

Шаги воспроизведения:

1.Ввести в поле user@example.com

2.Нажать кнопку «Проверить»

Ожидаемый результат:
Появился alert с текстом «Прекрасный email-адрес»

2.Название: Ввод email с цифрами

Шаги воспроизведения:
1.Ввести в поле user123@test123.com
2.Нажать кнопку «Проверить»
Ожидаемый результат:
Появился alert с текстом «Прекрасный email-адрес»


3. Название: Ввод email со специальными символами из разрешенного списка

Шаги воспроизведения:
1.Ввести в поле test!#$%@example.com
2.Нажать кнопку «Проверить»
Ожидаемый результат:
Появился alert с текстом «Прекрасный email-адрес»

Блок 3. Повторная валидация
13.Название: Повторное нажатие кнопки для валидного email
Шаги воспроизведения:
1.Ввести user@example.com
2.Нажать кнопку «Проверить»
3.Не изменяя значение, снова нажать кнопку «Проверить»
Ожидаемый результат:
При каждом нажатии появляется alert с текстом «Прекрасный email-адрес»
